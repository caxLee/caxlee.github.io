你的Agent电脑助手正在踩雷！最新研究揭秘Computer-Use Agent的安全漏洞

Jul 01, 2025
1 minute read

摘要：上海 AI Lab、中国科学技术大学和上海交通大学联合推出了CUA安全测试基准RiOSWorld，旨在评估智能助手在电脑使用中的安全风险。研究发现，大多数基于MLLM的CUA缺乏风险意识，容易受到网络诈骗和恶意操作的影响，安全性存疑。

主题标签：人工智能安全、电脑使用智能助手、风险评估
